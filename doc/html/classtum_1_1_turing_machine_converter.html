Detailed Description

Class for converting a given tape to an encoded tape.

The original tape is the tape from an arbitrary Turing Machine. There are 4 encoded tapes: Tape 1: The simulation tape. Tape 2: The encodation of the original TM. Tape 3: The current state. Tape 4: The working tape (scrap tape).
